9.8 Fancy Inspector

An alternative to default inspector is provided by the package `slime-fancy-inspector'. This inspector knows a lot about CLOS objects and methods. It provides many “actions” that can be selected to invoke Lisp code on the inspected object. For example, to present a generic function the inspector shows the documentation in plain text and presents each method with both a hyperlink to inspect the method object and a “remove method” action that you can invoke interactively. The key-bindings are the same as for the basic inspector (see Inspector).
