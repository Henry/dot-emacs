9.14 Highlight Edits

slime-highlight-edits is a minor mode to highlight those regions in a Lisp source file which are modified. This is useful to quickly find those functions which need to be recompiled (whith C-c C-c)

M-x slime-highlight-edits-mode
Turns slime-highlight-edits-mode on or off.
