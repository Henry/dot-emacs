9.15 inferior-slime-mode

The inferior-slime-mode is a minor mode is intended to use with a comint buffer. It provides some of the SLIME commands, like symbol completion and documentation lookup. To install it, add something like this to user .emacs: 

     (slime-setup '(inferior-slime-mode))
     (add-hook 'inferior-lisp-mode-hook (lambda () (inferior-slime-mode 1)))
M-x inferior-slime-mode
Turns inferior-slime-mode on or off.

The variable inferior-slime-mode-map contains the extra keybindings.
